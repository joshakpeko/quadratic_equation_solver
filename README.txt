************** QUADRATIC EQUATION SOLVER MINI PROGRAMM ****************

This is a little python interactive program that help solve 

quadratic equations like ax² + bx + c = 0.

Just provide inputs the program asks and it will pretty-print out

solutions of the equation.

This program leverages the power of python module `sympy` for pretty

mathematic symbols outputs.


Just enjoy !!!!!
